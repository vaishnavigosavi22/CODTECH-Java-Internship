
# Task 4: Recommendation System

## CODTECH Internship Task

This project implements a smart content-based recommendation system
using Java. The system analyzes user interests and recommends relevant
products based on feature matching and scoring.

### Features
- Accepts multiple user interests
- Uses scoring-based recommendation logic
- Ranks items based on relevance
- Console-based interactive system

### Technologies Used
- Java
- Collections Framework
- Basic AI recommendation logic
- NetBeans IDE

### How It Works
1. User enters interests
2. System compares interests with item features
3. Items are scored and ranked
4. Top recommendations are displayed

### How to Run
1. Open project in NetBeans
2. Run RecommendationApp.java
3. Enter interests when prompted
