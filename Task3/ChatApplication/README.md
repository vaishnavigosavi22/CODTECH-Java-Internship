
# Task 3: Java Client-Server Chat Application

## CODTECH Internship Task

This project implements a multi-client chat application
using Java Sockets and Multithreading.

### Features
- Server handles multiple clients
- Real-time message broadcasting
- Uses threads for each client

### Technologies
- Java
- Socket Programming
- Multithreading
- NetBeans IDE

### How to Run
1. Run ChatServer.java
2. Run ChatClient.java in multiple terminals
3. Start chatting
