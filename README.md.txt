# CODTECH Java Internship Tasks

This repository contains all four Java internship tasks completed
as part of the CODTECH Internship Program.

## Folder Structure

- Task1: File Handling Utility
- Task2: REST API Client
- Task3: Client-Server Chat Application
- Task4: Recommendation System

## Tasks Description

### Task 1: File Handling
Java program to read, write, and modify text files.

### Task 2: REST API Client
Java application that parses JSON data and displays it in a
structured format. Offline JSON is used due to network restrictions.

### Task 3: Chat Application
Client-server chat application using Java sockets and multithreading.

### Task 4: Recommendation System
Content-based recommendation system using user preferences and
scoring logic.

## Technologies Used
- Java
- NetBeans IDE
- Socket Programming
- JSON Parsing
- GitHub
